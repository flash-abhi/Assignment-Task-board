# Task Management App

 Task Management Application built using **React.js** and **Tailwind CSS**.

This project allows users to manage tasks efficiently using drag-and-drop functionality, priority filtering, searching, sorting, and persistent storage.

---

## Features

### Core Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Drag and drop between columns (Todo â†’ Doing â†’ Done)
- Persistent storage using LocalStorage

###  Advanced Features
-  Search tasks by title or description
-  Filter tasks by priority (Low, Medium, High)
-  Sort tasks by due date (Ascending / Descending)
-  Responsive design
